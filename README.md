## 🚀 Proyectos Destacados (Featured Projects)

### 🏢 1. Reception & Visitor Management System
Aplicación web orientada a la automatización del control de acceso corporativo, incorporando monitores de estatus en tiempo real, análisis automatizado de tiempos de respuesta del personal y exportación logística a PDF.

* **Stack:** PHP / Laravel, React, Tailwind CSS, MySQL.
* **Características Clave:** Monitoreo dinámico de estatus, asistente para detección de cuellos de botella y reportabilidad directiva.

### 🖥️ 2. IT Asset Management System
Sistema web de nivel empresarial diseñado para el control, auditoría e inventario logístico de activos tecnológicos y asignaciones de equipo de cómputo en entornos corporativos multi-plaza.

* **Stack:** PHP / Laravel, MySQL, JavaScript, Bootstrap / Tailwind CSS.
* **Características Clave:** Gestión jerárquica de plazas/áreas, deslinde automático de responsabilidades físicas de hardware y automatización de reportes corporativos.

### 🚗 3. GPS Installation & Vehicle Catalog System
Sistema web orientado a la gestión técnica de flotas, incorporando un buscador interactivo de fichas automotrices, diagramas de cableado eléctrico y la administración paso a paso de guías de instalación multimedia.

* **Stack:** PHP / Laravel, JavaScript, Tailwind CSS, MySQL.
* **Características Clave:** Filtrado predictivo por marca/modelo/año, CRUD para secuencias y manuales operativos, y control de acceso seguro para personal técnico.
