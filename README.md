## 🚀 Proyecto Destacado (Featured Project)

### 🖥️ [IT Asset Management System](https://github.com/luisgab98/it-asset-management-system)
Sistema web de nivel empresarial diseñado para el control, auditoría e inventario logístico de activos tecnológicos y asignaciones de equipo de cómputo en entornos corporativos multi-plaza.

- **Stack:** PHP / Laravel, MySQL, JavaScript, Bootstrap / Tailwind CSS.
- **Características:** Panel de asignaciones con buscador y paginación, relaciones jerárquicas complejas (Plazas ➡️ Jefes ➡️ Empleados 🔄 Equipos), exportación avanzada a Excel/PDF y generación automatizada de documentos de responsivas.
- **👉 [Ver Repositorio Completo con Capturas de Pantalla aquí](https://github.com/luisgab98/it-asset-management-system)**
